# Employee_Management_System_using_SQL_Python

## Project Overview

The Employee Management System is a database-driven application developed using Python and MySQL. The project is designed to manage employee records efficiently by performing CRUD (Create, Read, Update, Delete) operations. It also includes data analysis and export functionalities using Pandas.

This project demonstrates practical implementation of:
- Python programming
- SQL database management
- MySQL connectivity
- Data processing using Pandas
- Structured data handling

---

# Project Workflow

```text
Python Application
       ↓
MySQL Database
       ↓
Pandas Analysis
       ↓
CSV/Excel Export
```

---

# Features

- Add new employee records
- View employee details
- Update employee salary information
- Delete employee records
- Search employee data
- Department-wise salary analysis
- Export employee data to CSV and Excel
- Integration of Python with MySQL database
